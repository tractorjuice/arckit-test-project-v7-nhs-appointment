Get Started in 5 Minutes

1. Clone the Repository

git clone https://github.com/tractorjuice/arc-kit.git cd arc-kit

2. Choose Your AI Assistant

ArcKit works with Claude Code, OpenAI Codex CLI, or Gemini CLI. Commands are ready to use in:

  • .claude/commands/ - For Claude Code users
  • .codex/prompts/ - For Codex CLI users (set CODEX_HOME)
  • .gemini/commands/arckit/ - For Gemini CLI users

3. Start with a Plan

# Claude Code /arckit.plan Create project plan for cloud migration with 6-month timeline # Codex CLI /prompts:arckit.plan Create project plan for cloud migration with 6-month timeline # Gemini CLI arckit plan Create project plan for cloud migration with 6-month timeline

4. Follow the Workflow

Your project plan will show when to run each command:

  • Discovery: Stakeholders → Risk → Business Case
  • Alpha: Requirements → Data Model → Vendor Procurement
  • Beta: Design Reviews → Traceability → Compliance
  • Live: Deployment → Analysis → Benefits Realization
